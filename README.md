# overflow-guard

Monorepo for the `overflow-guard-react` package and the example website.

## Workspaces

- `packages/overflow-guard-react`: publishable React library
- `packages/overflow-guard-html`: reserved for the future custom element package
- `apps/website`: example SPA and docs site

## Commands

```sh
bun install
bun run build
bun run test
```

Build the website only with:

```sh
bun run build:website
```

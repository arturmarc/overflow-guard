# `<OverflowGuard>`

Build around content, not breakpoints.

Overflow Guard helps components adapt to their content instead of the viewport. No media or container queries. No magic pixel values.

![Overflow Guard demo](./docs/assets/overflow-guard-demo.gif)

Repository: <https://github.com/arturmarc/overflow-guard>
Website: <https://overflow-guard.vercel.app/>

## Libraries

- [`overflow-guard-react`](./packages/overflow-guard-react/README.md): React library for adapting UI when content stops fitting
- [`overflow-guard-html`](./packages/overflow-guard-html/README.md): HTML/custom element package readme

## Commands

```sh
bun install
bun run build
bun run test
```

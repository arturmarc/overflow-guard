# overflow-guard

Monorepo for the `overflow-guard-react` package and the example website.

Repository: <https://github.com/arturmarc/overflow-guard>

## Package READMEs

- [`packages/overflow-guard-react`](./packages/overflow-guard-react/README.md): React library docs
- [`packages/overflow-guard-html`](./packages/overflow-guard-html/README.md): future custom element package docs

## Workspaces

- [`packages/overflow-guard-react`](./packages/overflow-guard-react/README.md): publishable React library
- [`packages/overflow-guard-html`](./packages/overflow-guard-html/README.md): reserved for the future custom element package
- [`apps/website`](./apps/website/README.md): example SPA and docs site

## Commands

```sh
bun install
bun run build
bun run test
```

Build the website only with:

```sh
bun run build:website
```
